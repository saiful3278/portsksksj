## Overview

A practical Web app built for my village community. The goal is simple: make important local information easy to access and manage without hype or complexity.

## Features

- News and official notices
- Local directory (people, services, contacts)
- Events and updates
- Media gallery for photos and videos
- Admin update system for trusted editors

## Tech

- Android (Android Studio)
- Java/Kotlin
- Firebase (Auth, Database, Storage)
- REST APIs

## Outcome

Delivers reliable access to local information and reduces manual coordination. Built for real-world use and maintained with straightforward workflows.
